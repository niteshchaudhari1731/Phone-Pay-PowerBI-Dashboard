# 📊 PhonePe Power BI Dashboard

## 🚀 Project Overview
This Power BI dashboard provides an interactive analysis of PhonePe transaction data across India. The dashboard helps users understand transaction trends, user engagement, and regional performance through dynamic visualizations.

## 🎯 Objectives
- Analyze PhonePe transaction patterns.
- Compare state-wise performance.
- Track user growth and transaction volume.
- Identify key business insights through data visualization.

## 📈 Dashboard Features
✔ Total Transactions Analysis

✔ Transaction Amount Trends

✔ State-wise Performance Comparison

✔ User Growth Insights

✔ Interactive Filters and Slicers

✔ Dynamic Charts and KPIs

## 🛠️ Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Cleaning & Transformation
- Excel / CSV Dataset
